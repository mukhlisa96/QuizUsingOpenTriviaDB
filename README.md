# Quiz Game (Python – OOP)

A Python quiz game built using **Object-Oriented Programming**. The game presents True/False questions, checks user answers, and keeps track of the score until all questions are completed.

Quiz questions can be sourced from **Open Trivia Database**:
[https://opentdb.com/](https://opentdb.com/)

## Python Concepts Used

* Object-Oriented Programming (OOP)
* Classes & objects
* Constructors (`__init__`)
* Modules & imports
* Lists & dictionaries
* Variables & variable scope
* Loops
* Conditional statements
* User input & output
* Score tracking and game logic

## Project Structure

* `main.py` – creates questions and runs the quiz
* `question_model.py` – Question class
* `quiz_brain.py` – quiz logic and score handling
* `data.py` – question data
``

## Features

* True/False questions
* Automatic score calculation
* Final score display
* Easily extendable question set
